# Linux Study Notes - User Operations

## Basic Operations

### Adding a User

```shell
useradd -m username
```

### Setting Password

```shell
passwd username
```

### Deleting a User

```shell
userdel -r username
```

### Deleting User's Directory

```shell
rm -rf username
```

### Switching to Another User

```shell
su username
```
