# الأحرف المنقولة

| التسلسل المنقول | الحرف |
| -------- | ------------------------------------------ |
| `\b`     | حرف العودة |
| `\f`     | حرف التغيير |
| `\n`     | حرف السطر الجديد |
| `\r`     | حرف العودة للبداية |
| `\t`     | حرف التبويب |
| `\unnnn` | حرف يونيكود |
| `\xnn`   | حرف ASCII ، يتم تحديد رمز الحرف بواسطة الأرقام الست عشرية nn |
| `\'`     | علامة اقتباس مفردة |
| `\"`     | علامة اقتباس مزدوجة |
| `\\`     | حرف الشرطة الخلفية الواحدة |
