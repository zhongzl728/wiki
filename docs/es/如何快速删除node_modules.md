# Cómo eliminar rápidamente node_modules

Instala globalmente `rimraf`:

```shell
npm install rimraf -g
```

Ve al directorio raíz que contiene la carpeta `node_modules` y ejecuta el siguiente comando:

```shell
rimraf node_modules
```

¡Y eso es todo!
