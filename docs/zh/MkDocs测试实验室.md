## Mermaid

```mermaid
graph TD
    a1("Collect Voltage vs. QUOT data across skew and temperature")
    a2("Select RO and define Target QUOT (fix value)" )
    a1-->a2
```

```mermaid
graph LR
    A[Square Rect] -- Link text --> B((Circle))
    A --> C(Round Rect)
    B --> D{Rhombus}
    C --> D
```

```mermaid
graph LR
  A[Start] --> B{Error?};
  B -->|Yes| C[Hmm...];
  C --> D[Debug];
  D --> B;
  B ---->|No| E[Yay!];
```
